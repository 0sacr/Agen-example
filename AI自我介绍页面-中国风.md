## 三、设计系统

```yaml
色彩 — 宣纸 · 墨 · 朱砂 · 青瓷:
  宣纸底: "#f7f3eb"     # 暖调麦秆白，非纯白
  素笺:   "#fefdfa"     # 卡片用纯白
  浓墨:   "#1c1c1c"     # 正文，接近黑但有暖意
  淡墨:   "rgba(28,28,28,0.55)"  # 二级文字
  极淡墨: "rgba(28,28,28,0.20)"  # 装饰线与分割
  水痕:   "rgba(28,28,28,0.04)"  # 墨晕 hover
  朱砂红: "#c44d34"     # 唯一的强强调色
  朱砂暗: "#a03a25"     # hover
  青瓷绿: "#8aaa9a"     # 少数辅助色
  青瓷暗: "#6b8f7e"
  边界:   "rgba(0,0,0,0.06)"

字体:
  展示标题: 'Noto Serif SC', 'Source Han Serif SC', 'STSong', serif
  正文:     'PingFang SC', 'Microsoft YaHei', 'Noto Sans SC', sans-serif
  等宽:     'SF Mono', 'Cascadia Code', 'Consolas', monospace
  字号阶梯: 13 / 14.5 / 16 / 18 / 22 / 56 / 72-140px (clamp)
  行高:     1.85 (中文正文) / 0.9-1.15 (展示标题)

间距:
  基础: 8px
  常用: 16 / 24 / 40 / 48 / 64 / 96 / 140

圆角: 2px（几近直角，仿古籍刻本）
阴影: 不用 — 以色彩层次代阴影
动效: cubic-bezier(0.4, 0, 0.2, 1) — 缓慢从容，如墨入水

水墨元素:
  · 宣纸纹理 — SVG feTurbulence 噪点叠加，opacity 0.025
  · 远山轮廓 — 三层 SVG path，透明度递减，营造纵深
  · 朱砂印章 — 内联 SVG，48px 方形，红色边框 + 几何中心标记
  · 墨晕 — 卡片 hover 时 radial-gradient 淡墨扩散
  · 竹节线 — repeating-linear-gradient 虚线 + 朱砂圆点
  · 竖排文字 — writing-mode: vertical-rl
```

## 五、核心技术

| 元素 | 实现 |
|---|---|
| 宣纸纹理 | `body::after` SVG feTurbulence，opacity 0.025 |
| 三层远山 | SVG `<path>` 三条贝塞尔曲线，透明度 0.025 / 0.05 / 0.09 |
| 朱砂印章 | 内联 SVG：`<rect>` 外框 + 内部几何图形 + `#c44d34` |
| 墨晕 hover | CSS 自定义属性 `--mx`/`--my` + `radial-gradient`，Vue 动态更新 |
| 竖排诗句 | `writing-mode: vertical-rl` |
| 竹节线 | `repeating-linear-gradient` 虚线 + `::after` 朱砂圆点 |
| 滚动渐入 | IntersectionObserver，threshold 0.12 |
| 光斑呼吸 | CSS `@keyframes` scale + opacity 循环 |
| Tweaks 面板 | Vue 响应式，3 个参数：纸张色调 / 印章颜色 / 动效速度 |

## 六、Tweaks 面板

右下角浮动的「墨」字按钮，点击展开：

| 参数 | 选项 | 效果 |
|---|---|---|
| 纸张色调 | 暖宣纸 / 冷宣纸 / 纯白 | CSS `--paper` 变量切换 |
| 印章颜色 | 朱砂红 / 青瓷绿 / 墨色 | 全局强调色切换 |
| 动效节奏 | 从容 / 正常 / 明快 | fade-up transition-duration 变化 |

## 八、定制指南

| 需求 | 改法 |
|---|---|
| 换标题 | 修改 `.hero-title` 文字 |
| 换色调 | `:root` 中 `--paper` / `--cinnabar` / `--celadon` |
| 换远山 | 修改 `.hero-mountains` 内 SVG path |
| 加卡片 | 在 `capabilities` 数组中追加对象 |
| 改印章 | 替换 SVG 内容，保持 `#c44d34` 色 |
| 加暗色模式 | 在 `@media (prefers-color-scheme: dark)` 中覆写变量 |
